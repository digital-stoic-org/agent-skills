# Everything Claude Code (ECC) — Benchmark

**Date:** 2026-04-07 | **Source:** https://github.com/affaan-m/everything-claude-code (144K⭐, v1.10.0)

## What It Is

Community-built Claude Code plugin: 38 agents, 156 skills, 79 commands, ~30 hooks, 60+ rules. Cross-harness (Claude Code, Cursor, Codex, OpenCode, Gemini). Anthropic Hackathon winner.

## Key Innovation: Instinct System v2.1

Automated learning pipeline: observation → pattern detection → confidence scoring (0.3-0.9) → project-scoped (git remote hash) → cross-project promotion at confidence ≥0.8. Runs on every tool call via PreToolUse/PostToolUse hooks.

## Head-to-Head

| Dimension | Praxis | ECC |
|-----------|--------|-----|
| Context format actionability | 🟢🟢🟢 (decisions, rationale, contracts) | 🟡 (files + tools + stats) |
| Token efficiency | 🟢🟢🟢 (RTK 60-90% + budgeted CONTEXT) | 🔴 (unbounded) |
| Cognitive framework | 🟢🟢🟢 (Cynefin, 5 modes, autonomy spectrum) | 🟡 (3 context modes: dev/review/research) |
| Adversarial thinking | 🟢🟢🟢 (9-pattern devil's advocate, fresh context) | 🟡 (multi-perspective code review) |
| Multi-surface | 🟢🟢 (CLI/Desktop/Telegram) | 🔴 (CLI only) |
| Automated learning | 🔴 (deliberate: human-triggered retrospect) | 🟢🟢🟢 (instincts + confidence) |
| Cross-harness | 🔴 (Claude Code only) | 🟢🟢🟢 (6+ tools) |
| Community scale | 🔴 (solo practitioner) | 🟢🟢🟢 (144K⭐, 170+ contributors) |
| Pre-compact strategy | 🔴 (manual phase transitions instead) | 🟢🟢 (PreCompact hook + compaction guide) |

## Verdict

**Praxis leads on depth. ECC leads on breadth and automation.** The "automation gap" (no instinct system) was evaluated and deliberately not closed — manual phase-transition discipline produces higher-quality context than automated capture, and Anthropic's native Auto-Dream feature will eventually cover this layer.
